## Install

```bash
# using npm
npm install

# using pnpm
pnpm install

# using yarn
yarn install
```

## Run it

```bash
# using npm
npm run dev

# using pnpm
pnpm dev

# using yarn
yarn dev
```

## Result

Open your browser and visit the web page

- `http://localhost:3000/api/docs` shows the UI
- `http://localhost:3000/api/docs/open-api` serves the JSON.
